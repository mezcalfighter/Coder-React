// 1 primer forma: Utilizando la librebria create-react-app

1.- descargar libreria:
    npm i -g create-react-app
    npm i --global create-react-app

2.- Crear la app con create-react-app:
    create-react-app [nombre-app]
    ej. create-react-app mi-primer-app

3.- Cambiar de directiorio
    cd miPrimerApp

4.- Correr proyecto
    npm start

// 2 segunda forma: usando npx
    npx create-react-app [nombre-app]
    ej. npx create-react-app mi-primer-app
    cd mi-primer-app
    npm start

// 3 tercera forma: Vite
    npm create vite@latest [nombre-app]
